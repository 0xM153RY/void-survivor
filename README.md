# VOID SURVIVOR

A retro arcade bullet-hell arena shooter built entirely with vanilla HTML5 Canvas, JavaScript, and the Web Audio API. No external assets or libraries.

**Play it here:** [https://0xm153ry.github.io/void-survivor/](https://0xm153ry.github.io/void-survivor/)

## Features

- Procedurally generated synthwave soundtrack that intensifies with gameplay
- 6 enemy types with unique behaviors (Drifter, Dasher, Swarm, Orbiter, Sprayer, Bomber)
- Boss fights every 5 waves
- 6 powerup types (Rapid Fire, Spread Shot, Shield, Speed Boost, Bomb, Extra Life)
- Full menu system with settings, high scores, how-to-play, and credits
- Persistent settings and high score tracking via localStorage
- Mobile touch controls with virtual joystick and auto-aim
- CRT-style visual effects (scanlines, vignette, screen shake)

## Controls

### Desktop
- **WASD / Arrow Keys** - Move
- **Mouse** - Aim
- **Left Click / Space** - Shoot
- **Right Click / Shift** - Bomb

### Mobile
- **Left thumb** - Virtual joystick to move
- **Right side tap** - Aim direction
- **Auto-fire** enabled
- **Bomb button** - Screen clear

## Running Locally

Open `index.html` in any modern browser. No build step or server required.
